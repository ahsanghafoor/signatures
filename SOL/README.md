## How to Use

### For Outlook (Desktop)

1. Open the `signature.html` file in your browser
2. Select all content (Ctrl+A / Cmd+A)
3. Copy it (Ctrl+C / Cmd+C)
4. Open Outlook, go to **File** > **Options** > **Mail** > **Signatures**
5. Click **New** to create a new signature
6. Paste (Ctrl+V / Cmd+V) the copied signature into the editor
7. Click **OK** to save

### For Gmail

1. Open the `signature.html` file in your browser
2. Select all content (Ctrl+A / Cmd+A)
3. Copy it (Ctrl+C / Cmd+C)
4. Go to Gmail Settings (gear icon) > **See all settings** > **General**
5. Scroll down to **Signature** section
6. Click **Create new** and give it a name
7. Paste (Ctrl+V / Cmd+V) the copied signature
8. Scroll down and click **Save Changes**

### For Apple Mail

1. Open the `signature.html` file in your browser
2. Select all content (Ctrl+A / Cmd+A)
3. Copy it (Ctrl+C / Cmd+C)
4. Open Mail, go to **Mail** > **Preferences** > **Signatures**
5. Click the **+** button to create a new signature
6. In the signature editor, paste (Cmd+V) the copied signature
7. Close the preferences window to save
